# Security Policy

## 方針 (NF-008)

- KuruVoice は初期バージョンでは **外部通信を行いません**。
- 設定ファイルは **TOML のみ** を読み込み、任意コード実行につながる形式（実行可能スクリプト等）は使用しません。
- オーディオデータはローカルでのみ処理され、外部送信されません。

## 脆弱性の報告

セキュリティ上の問題を見つけた場合は、公開 Issue ではなく、メンテナへ非公開で連絡してください。
できるだけ以下を含めてください:

- 影響範囲・再現手順
- 対象バージョン / OS
- 想定される影響

確認後、修正版をリリースします。

## 利用上の注意 (NF-009)

本ソフトは本人の声の補正・配信品質向上のためのものです。
なりすまし・詐欺・本人確認突破・無断録音などへの利用は禁止です。詳細は `docs/safety.md`。
