# セーフティポリシー

KuruVoice は、**自分の声の補正・加工・配信品質向上** を目的としたツールです。

## 禁止・非推奨事項 (NF-009)

- 他人へのなりすまし目的での利用
- 詐欺・迷惑行為での利用
- 本人確認（音声認証など）の突破
- 無断録音・盗聴
- 特定人物の声の無断再現

## 設計上の安全策

- **リミッターを最終段に固定**（NF-005）。過大音量・音割れを防ぎ、聴覚を保護。
- 外部通信を行わない（NF-008）。音声・設定はローカルのみで扱う。
- 設定は TOML のみ。任意コード実行につながる形式は読み込まない。

## 録音について

`--record-test` はローカルに WAV を書き出すだけで、外部送信は行いません。
他者の声を録音・利用する場合は、必ず本人の同意を得てください。
